Container

Default Container

This default container will set max-width to each break points

Extra Small: 100% if screen size is lesser than 576px

Small: 540px if screen size is greater than or equal to 576px

Medium: 720px if screen size is greater than or equal to 768px

Large: 960px if screen size is greater than or equal to 992px

X-Large: 1140px if screen size is greater than or equal to 1200px

XX-Large: 1320px if screen size is greater than or equal to 1400px

Code:

                                    
                                        <div class="container">
    <p class="text-white">This default container will set max-width to each break points</p>
                                            // Explaining Sizes of xs, s, md, lg, xl, xxl
</div>

Responsive Container

This small container have it's content change size to 100% if equal to 576px
This medium container have it's content change size to 100% if equal to 768px
This large container have it's content change size to 100% if equal to 992px
This extra large containxl have it's content change size to 100% if equal to 1200px
This x-extra large container have it's content change size to 100% if equal to 1400px

                                        <div class="container-sm bg-black text-white">
    This small container have it's content change size to 100% if equal to 576px
</div>
                                        <div class="container-mg bg-black text-white">

</div>

Fluid Container

This is a fluid container with 100% at every breakpoints.
