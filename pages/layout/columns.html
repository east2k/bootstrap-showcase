Reodering Columns 1

first .col No order applied
second .col 5th or largest order applied
third .col 2nd order applied
fourth .col 3rd order applied
fifth .col 1st order applied

Code:

                                    
                                        <div class="container">
    <div class="row">
        <div class="col">
            first .col No order applied
        </div>
        <div class="col order-5">
            second .col 5th or largest order applied
        </div>
        <div class="col order-2">
            third .col 2nd order applied
        </div>
        <div class="col order-3">
            fourth .col 3rd order applied
        </div>
        <div class="col order-1">
            fifth .col 1st order applied
        </div>
    </div>
</div>
