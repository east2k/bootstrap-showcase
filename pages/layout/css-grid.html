Adding Rows

g-start-(amount)

style="grid-row: (amount)"

no class
.g-start-2 with style="grid-row: 2"
.g-start-3 with style="grid-row: 3"

Code:

                                    
                                        <div class="grid" style="--bs-columns: 10; --bs-gap: 1rem">
    <div>no class</div>
    <div class="g-start-2" style="grid-row:2">.g-start-2 with style="grid-row: 2"</div>
    <div class="g-start-3" style="grid-row:3">.g-start-2 with style="grid-row: 3"</div>
</div>
